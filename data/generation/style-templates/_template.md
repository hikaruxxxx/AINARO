# 文体テンプレート(共通仕様)

## 目的

各サブジャンルの `{subgenre_id}.md` は、Layer 5(ep1本文)生成時に LLM へ渡す **文体ガイド**。
人称・テンポ・描写密度・語彙傾向などジャンル固有の文体特性を定義する。

## 必須セクション

```markdown
# {サブジャンル名}

## 人称
- 一人称 / 三人称 / 主語省略型
- なぜそうするかの理由

## テンポ
- 速い / 中 / 遅い
- 会話比率の目安

## 描写密度
- 心情描写の比重
- 情景描写の比重
- アクション描写の比重

## 語彙傾向
- 装飾的 / 簡潔 / 平易 / 専門的
- 避けるべき語彙

## 文体サンプル(50〜100字)
- 良い例
- 悪い例

## NG表現
- このジャンルで避けるべき書き方

## 字数配分(3500-4500字の場合)
- 各シーン(冒頭/展開/転機/引き)に何字割くか
```

## 運用ルール

- 1ファイル400〜600字程度
- サンプルは必ず付ける(LLMにとって最強の指示は具体例)
- batch 結果でテンプレ化が観測されたらすぐ更新
