# 文体プロファイル：廃村の先生と古代魔術

## パラメータ

```yaml
pov: third_person
tempo: slow
sentence_length_avg: 28
dialogue_ratio: 0.05
inner_monologue_ratio: 0.15
description_density: high
humor_level: low
tension_curve: wave
vocabulary_level: standard
line_break_frequency: frequent
```

## 文体の特徴

五感描写が極めて濃密。雨粒の感触、泥の匂い、鉄の味など身体感覚を執拗に重ねる。比喩が詩的で内省的（「雨の方が先に問いを立てている」等）。短文と長文のリズム交替が巧み。句読点の間で余韻を作り、沈黙や静けさそのものを描写対象にしている。
