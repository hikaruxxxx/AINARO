# 文体プロファイル：灰の館、七つの鐘

## パラメータ

```yaml
pov: first_person
tempo: slow
sentence_length_avg: 28
dialogue_ratio: 0.20
inner_monologue_ratio: 0.30
description_density: high
humor_level: low
tension_curve: wave
vocabulary_level: standard
line_break_frequency: frequent
```

## 文体の特徴

一人称で内面の揺れを丁寧に追う文体。五感描写が濃密で、甘い腐臭・ぬるい湿り気・微動だにしない蝋燭の火など、不自然な静止と異物感で恐怖を醸成する。短文と長文を織り交ぜ、思考の断片や擬音（ずるり）を独立行で配置しリズムを作る。比喩は具体的で体感的。
