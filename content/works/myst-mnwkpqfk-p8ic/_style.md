# 文体プロファイル：鷹森邸の声なき者たち

```yaml
pov: first_person
tempo: slow
sentence_length_avg: 28
dialogue_ratio: 0.20
inner_monologue_ratio: 0.30
description_density: high
humor_level: low
tension_curve: wave
vocabulary_level: standard
line_break_frequency: frequent
```

## 文体の特徴

一人称視点で心理描写が細やか。短文と体言止めを織り交ぜ、視覚・嗅覚・聴覚の五感描写で空気の重さを積み上げる。「——」による余韻、擬音の反復（ぴちゃん、ぴちゃん）で不穏さを醸成。感情を直接語らず、沈黙や所作の描写で語り手の孤独を滲ませる抑制的な文体。
