# 文体プロファイル：帳の目、探偵の目

## パラメータ

```yaml
pov: third_person
tempo: fast
sentence_length_avg: 18
dialogue_ratio: 0.15
inner_monologue_ratio: 0.15
description_density: medium
humor_level: low
tension_curve: wave
vocabulary_level: standard
line_break_frequency: frequent
```

## 文体の特徴

三人称視点で探偵の感覚と思考を密着描写。五感(臭気・音・体温・光)の積層で場の空気を立ち上げ、短文と体言止めで緊張を刻む。「観察」を動詞化した文体で、瞳孔・筋肉・重心など身体細部の観察描写が多い。比喩は即物的(心臓・サイレン)、地の文に探偵の独白が混じるハードボイルド調。
