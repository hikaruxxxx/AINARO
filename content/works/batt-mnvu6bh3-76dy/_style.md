# 文体プロファイル：共振破壊のガラス細工

```yaml
pov: first_person

description_density: medium

```

## 文体の特徴

一人称視点で内省と情景描写が緊密に絡む。比喩が具体的かつ感覚的で、「刃ごと手が震えている」等の体感的表現が多い。ランク社会の嘲笑と主人公の静かな意地が対比され、視覚・聴覚・触覚を束ねた没入描写が強い。短い台詞とモノローグの往復でテンポを作る。
