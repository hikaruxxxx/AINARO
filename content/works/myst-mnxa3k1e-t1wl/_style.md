# 文体プロファイル：鏡守の亡霊

## パラメータ

```yaml
pov: third_person
tempo: slow
sentence_length_avg: 28
dialogue_ratio: 0.05
inner_monologue_ratio: 0.15
description_density: high
humor_level: low
tension_curve: wave
vocabulary_level: standard
line_break_frequency: frequent
```

## 文体の特徴

五感描写が濃密で、特に痛覚・触覚・嗅覚を多用する体感型の文体。短文と長文を交互に配置し緩急をつける。内面独白が多く、回想と現在を映像的にカットバックさせる。比喩は硬質でハードボイルド調。句点で止める体言止めを効果的に使用。
