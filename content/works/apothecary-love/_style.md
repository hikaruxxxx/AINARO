# 文体プロファイル：宮廷薬師の秘密の処方箋

## パラメータ

```yaml
pov: third_person_limited       # 三人称限定（リリアーナ寄り）
tempo: medium                    # 基本テンポ: 中程度（仕事描写とロマンスのバランス）
sentence_length_avg: 23          # やや短め寄りの中文。テンポと描写のバランス
dialogue_ratio: 0.35             # 会話3.5割
inner_monologue_ratio: 0.20      # 内面独白2割（リリアーナの秘密への不安と薬草への情熱）
description_density: medium-high # 描写はやや厚め（薬草・調合シーンを丁寧に）
humor_level: moderate            # 薬草オタク暴走時のユーモア。基本は落ち着いたトーン
tension_curve: dual_track        # 仕事の緊張感とロマンスの緊張感が並行
vocabulary_level: refined        # 品のある語彙。薬学用語が自然に混ざる
line_break_frequency: moderate   # 改行は適度
```

## 文体の特徴

この作品の文体は「仕事の矜持」と「秘密の緊張感」の二重構造。薬草・調合シーンの専門的な描写がリリアーナの有能さを示し、その合間にガルシアとの距離感を丁寧に描く。

- 地の文: 落ち着いた筆致。薬草・調合の描写は五感を使って臨場感を出す
- 会話: リリアーナの敬語とガルシアの簡潔さの対比。仕事の会話から私的な会話へ変わる過程
- 独白: リリアーナの内面は二層。薬師としての冷静さと、秘密を抱える不安
- お仕事描写: 調合シーンは丁寧に。読者が「この仕事面白い」と思える描写を心がける

### 文体の参考例

```
　乳鉢の中で、乾燥させた月見草の花弁が粉になっていく。
　規則正しい円運動。力加減を誤れば有効成分が壊れる。リリアーナの手つきに迷いはなかった。
「……薬師」
　背後から低い声。手が止まりかける。止めない。調合中に手を止めるのは三流だ。
「騎士団長殿。調合室への無断立入は規則違反ですが」
「規則の話をするのか。身分を偽っている人間が」
　乳鉢を持つ指先に、わずかに力がこもった。それだけだ。振り返らない。
「……何の、ご用でしょうか」
```

## 話ごとのテンポ目安

- 第1話: fast（身バレのピンチ。テンポよく引き込む）
- 第2-5話: medium（秘密を共有した二人の距離感。仕事描写で魅力を見せる）
- 第6-10話: medium（信頼の構築。薬師としての腕が認められていく）
- 第11-15話: medium→fast（リリアーナの過去に迫る事件。正体露見の危機）
- 第16-20話: fast（宮廷の陰謀。薬学の知識が事件解決の鍵に）
- 第21-25話: medium→climax（身分を賭けた選択）
- 第26-30話: climax→medium（決着と新しい出発）
