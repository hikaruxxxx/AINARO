# 文体プロファイル：追放された聖女は冷徹公爵に溺愛される

## パラメータ

```yaml
pov: third_person_limited       # 三人称限定（セラフィーナ寄り）
tempo: medium                    # 基本テンポ: 中程度（じっくり溺愛を描く）
sentence_length_avg: 25          # やや長めの文で丁寧に描写
dialogue_ratio: 0.35             # 会話3.5割
inner_monologue_ratio: 0.20      # 内面独白2割（主人公の戸惑い描写）
description_density: high        # 描写は厚め（辺境の風景、溺愛の行動描写）
humor_level: low                 # コミカルさは控えめ。温かみのあるユーモア
tension_curve: gradual_rise      # 緩やかに感情が積み上がる
vocabulary_level: elegant        # 上品で柔らかい語彙
line_break_frequency: moderate   # 改行は適度
```

## 文体の特徴

この作品の文体は「不器用な優しさの描写」が核。ヴィクトルの溺愛は言葉ではなく行動で示し、セラフィーナの戸惑いを通じて読者にキュンを届ける。

- 地の文: 柔らかく端正。風景や季節の描写を丁寧に
- 会話: ヴィクトルは短文、セラフィーナは丁寧語。温度差がキュンを生む
- 独白: セラフィーナの内面は素直。戸惑いや気づきを自然に描く
- 溺愛描写: 「説明」しない。行動を描き、読者に気づかせる

### 文体の参考例

```
　朝食の席に、見慣れない小瓶が置いてあった。
　薄い琥珀色の液体。蓋を開けると、蜂蜜の甘い香りが広がる。
「……これは？」
「辺境の蜂蜜だ。喉に良い」
　ヴィクトルはそれだけ言って、視線を書類に戻した。
　セラフィーナは小瓶を両手で包んだ。昨夜、少し咳をしたのを——聞いていたのだろうか。
　頬が熱い。これは、朝の日差しのせいだ。きっとそうだ。
```

## 話ごとのテンポ目安

- 第1話: fast（追放、出会い。テンポよく引き込む）
- 第2-5話: medium（辺境での暮らし、距離が縮まる日常）
- 第6-10話: medium（溺愛の芽生え、セラフィーナの葛藤）
- 第11-15話: medium→fast（過去の脅威が迫る）
- 第16-20話: fast（教会との対立、ヴィクトルの過去）
- 第21-25話: medium→climax（関係の危機と再構築）
- 第26-30話: climax→medium（決着と大団円）
