# 文体プロファイル：雷鳴のバチ

## パラメータ

```yaml
pov: first_person
tempo: slow
sentence_length_avg: 28
dialogue_ratio: 0.05
inner_monologue_ratio: 0.30
description_density: high
humor_level: low
tension_curve: wave
vocabulary_level: standard
line_break_frequency: frequent
```

## 文体の特徴

一人称現在進行形の内省的な語り。五感描写が極めて精緻で、音・振動・匂い・温度を身体感覚として描く。短文と長文のリズム切り替えで緊張と弛緩を制御。比喩は体感に根ざし、感情を直接名指しせず身体反応で表現する抑制的な文体。
