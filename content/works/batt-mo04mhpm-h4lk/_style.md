# 文体プロファイル：灰翼の軍師

## パラメータ

```yaml
pov: third_person
tempo: slow
sentence_length_avg: 28
dialogue_ratio: 0.12
inner_monologue_ratio: 0.15
description_density: high
humor_level: low
tension_curve: wave
vocabulary_level: standard
line_break_frequency: frequent
```

## 文体の特徴

短文と体言止めを多用した硬質な文体。内面描写は感情を直接語らず身体感覚や動作で暗示する抑制的な筆致。情景描写は五感（泥の感触、煙草の匂い、蝋燭の揺れ）を通じて空気感を構築し、回想と現在を滑らかに往復させる構成力が特徴。
