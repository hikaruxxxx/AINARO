# 文体プロファイル：潮風と退職届

## パラメータ

```yaml
pov: first_person

dialogue_ratio: 0.20
inner_monologue_ratio: 0.35
description_density: high
humor_level: low
tension_curve: ascending
vocabulary_level: standard
line_break_frequency: frequent
```

## 文体の特徴

- 一人称「私」による内省的な語り。感情を直接名指しせず、身体感覚や比喩で間接的に伝える（「喉の奥に刺さって、抜けなくなった」「身体の奥から浮き上がってくるような感覚」）
- 短文と長文のリズム交替が特徴的。感情が高まる場面では意図的に短文を連打し、余韻を残す（「口が動かなかった。」「笑えなかった。」）
- 五感描写が濃密。視覚だけでなく触覚・嗅覚・聴覚を重ねて場面のリアリティを構築する（カーペットの繊維、消毒液のにおい、ビル風の唸り）
- 会話文は少なく、出てくるときは状況を一変させる鋭い一言として機能する（「このままだと入院です」「チームワークって知ってる？」）
- 時系列を微妙にずらす回想挿入で、現在の感情に過去の文脈を重ねるレイヤー構造を持つ
