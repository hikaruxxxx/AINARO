# 文体プロファイル：透明な観測者の航行日誌

## パラメータ

```yaml
pov: third_person
tempo: slow
sentence_length_avg: 28
dialogue_ratio: 0.10
inner_monologue_ratio: 0.15
description_density: high
humor_level: low
tension_curve: wave
vocabulary_level: standard
line_break_frequency: frequent
```

## 文体の特徴

数値や時刻を多用した精密な観測記録体。五感描写（冷気・匂い・振動）を丹念に重ね、視線の秒数や声量のデシベル差など微細な変化を定量的に記述する。感情を直接語らず、身体反応と行動の記録を通じて浮かび上がらせる抑制的な文体。
