# 文体プロファイル：魂契のテイマー

## パラメータ

```yaml
pov: third_person
tempo: fast
sentence_length_avg: 18
dialogue_ratio: 0.15
inner_monologue_ratio: 0.15
description_density: medium
humor_level: low
tension_curve: wave
vocabulary_level: standard
line_break_frequency: frequent
```

## 文体の特徴

五感描写が濃密で、特に触覚・嗅覚の描写が臨場感を生んでいる。短文と長文を意図的に切り替え、緊迫感のリズムを作る。内面描写は抑制的で、感情を直接名指しせず身体反応で語る。比喩は具体的かつ即物的。ダッシュや体言止めを効果的に使い、映像的なテンポを維持している。
