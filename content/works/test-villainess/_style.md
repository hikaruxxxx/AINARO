# 文体プロファイル：断罪前夜に目覚めた悪役令嬢

## パラメータ

```yaml
pov: first_person              # 一人称（リゼット視点）
tempo: fast                     # 基本テンポ: 速め
sentence_length_avg: 20         # 短文主体でテンポ重視
dialogue_ratio: 0.40            # 会話4割
inner_monologue_ratio: 0.30     # 内面独白3割（ツッコミ多め）
description_density: medium     # 描写は中程度
humor_level: high               # コミカル度高め
tension_curve: wave             # 緊張と弛緩の波
vocabulary_level: casual        # カジュアル寄り
line_break_frequency: frequent  # 改行多め（スマホ読み意識）
```

## 文体の特徴

この作品の文体は「外面と内面のギャップ」で読者を笑わせる構造。

- 地の文（外面描写）: 端正な表現。「令嬢」としての振る舞い
- 独白（内面）: くだけた現代語。ツッコミ、焦り、本音
- この二層のギャップが笑いを生む。独白は（）で囲む

「ごきげんよう、エレーヌ様」

## 話ごとのテンポ目安

- 第1話: fast（転生覚醒、状況把握、テンポよく引き込む）
- 第2-3話: fast（断罪回避の行動、緊張感を維持）
- 第4-5話: medium（日常回、キャラの魅力を見せる）
- 第6-7話: medium→fast（新たな問題発覚）
- 第8-10話: climax（第1幕クライマックス）
