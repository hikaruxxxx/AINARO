# 文体プロファイル：透明人間の観察ノート

```yaml
pov: first_person
tempo: slow

description_density: high
humor_level: low

```

## 文体の特徴

一人称独白体。短いセンテンスと比喩を織り交ぜ、教室の空気や光を触覚的に描写する。感情を名付けることを避け、観察と分析で距離を取る冷めた文体。地の文が長く、心理の解像度が高い一方、会話は短く引用的に挿入される。「期待は裏切られるために存在する」など格言的な一文で思考を締める癖が特徴。
