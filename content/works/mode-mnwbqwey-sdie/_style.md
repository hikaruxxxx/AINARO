# 文体プロファイル：季節の手紙

## パラメータ

```yaml
pov: first_person
tempo: slow
sentence_length_avg: 28
dialogue_ratio: 0.05
inner_monologue_ratio: 0.30
description_density: high
humor_level: low
tension_curve: wave
vocabulary_level: standard
line_break_frequency: frequent
```

## 文体の特徴

極めて繊細な感覚描写が特徴。視覚より触覚・嗅覚・聴覚を多用し、結露の冷たさや蝋梅の香りなど微細な身体感覚で心理を語る。短文と長文を交互に配置するリズム。説明を排し、描写の積み重ねで感情を浮かび上がらせる抑制的な文体。
