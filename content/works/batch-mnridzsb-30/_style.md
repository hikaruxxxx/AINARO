# 文体プロファイル：封印令嬢、辺境で最強の楽園を創る

```yaml
pov: first_person
tempo: medium

description_density: medium

```

## 文体の特徴

一人称現在の内省的な語りと、視覚・嗅覚・触覚を緻密に描く情景描写が特徴。魔灯の光、絹の衣擦れ、薔薇の香水、古紙の匂いなど五感の細部を積層させる。仮面の下の感情を短い独白で切り取り、静謐な緊張感を維持する。比喩は抑制的で品がある。
