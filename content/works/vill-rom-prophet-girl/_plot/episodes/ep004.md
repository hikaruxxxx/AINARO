# ep004「呪いの令嬢と赤毛の騎士」

## コンテキストプロファイル: daily_life

## シーン構成

### シーン1: 噂と日常（宮廷の廊下→アイリーンの自室）
- 場所: 宮廷の廊下、離宮の自室
- 時間: 午前
- 登場: アイリーン, リオン, ルル, 宮廷の人々（モブ）
- 目的: 護衛生活の日常を描き、二人の距離が自然に縮まる様子を見せる
- 文字数目安: 2,200字

#### 感情の弧
噂への苛立ち → リオンの素朴さへの戸惑い → 温かさへの抵抗 → ほのかな安心

#### 五感描写の指定
- 必須: 嗅覚「リオンが持ってきた野花の匂い」
- 推奨: 触覚「紅茶のカップの温もり」

### シーン2: 予言の意味（離宮のバルコニー・夕刻）
- 場所: 離宮のバルコニー
- 時間: 夕刻
- 登場: アイリーン, リオン
- 目的: 「あなたは必ず私を愛する」という予言に縛られているのでは？という疑念を芽生えさせる
- 文字数目安: 1,800字

#### 感情の弧
穏やかな会話 → 予言への疑問が浮上 → 自分の気持ちへの戸惑い

#### このシーンで設置する伏線
- 設置: F018 「予言がなくても傍にいてくれるか」というアイリーンの内なる問い

## 引きの設計
アイリーンが心の中で「もし予言がなくても、あなたはわたくしの傍にいてくれるの？」と問う。答えは出ない。
