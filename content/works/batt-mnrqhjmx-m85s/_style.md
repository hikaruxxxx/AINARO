# 文体プロファイル：命燃拳の運び屋

```yaml
pov: first_person
tempo: fast

description_density: medium
humor_level: low

```

## 文体の特徴

一人称視点の短文畳みかけで緊張感を演出。五感描写が濃密で、血・腐臭・蛍光灯の明滅など視覚と嗅覚を重ねて不穏さを構築する。体感温度や痛覚を通じて身体感覚に訴える筆致。荒んだ独白と比喩的表現が混在し、主人公の諦観と若さが同居する語り口。擬音と体言止めでテンポを刻む。
