# 第1話「7回目のおはよう」

## シーン構成

### シーン1: 覚醒（アリシアの寝室・朝）

- 場所: ノワール公爵邸、アリシアの寝室
- 登場: アリシア
- 目的: 7回目のループ開始。冒頭から「また始まった」感と疲労を全開で見せる。読者を一気に引き込む
- 感情の弧: 虚無 → ため息 → 「……もういいか」という諦めの境地
- 描写ポイント:
  - 見飽きた天蓋付きベッド。豪華だが7回目には感動ゼロ
  - 鏡の中の10歳の自分。目が死んでいる。「（この目、前世の月曜朝より酷い）」
  - 窓の外の朝日。「（何回見ても眩しい。ていうか7回見た）」
  - 身体を起こす動作が異様にだるい
- 冒頭フック: 一行目で「また始まった」感を出す。例: 「——7回目の朝日は、やっぱり眩しかった。」

### シーン2: 過去の走馬灯（独白・回想）

- 場所: 同上（ベッドの上で天井を見つめながら）
- 登場: アリシア（独白中心）
- 目的: 6回のループの失敗をダイジェストで回想。読者にルールと設定を伝える。コメディで処理
- 感情の弧: 苦い記憶 → でも笑えるくらいに達観 → 「何やっても無駄だった」
- 描写ポイント:
  - 各周回の失敗をテンポよく列挙（一つ一つは短く）
  - 「1周目: 何も知らなかった。断罪された。2周目: 完璧令嬢で攻めた。完璧すぎて怖いと言われた。——は？」
  - 6周目の「ルークの目が揺れた」瞬間だけ、少しトーンが変わる
  - 最後は「もう、いいかな」とため息
- ペース: 高速。箇条書き的なリズムで6回を駆け抜ける

### シーン3: 宣言（朝食の間）

- 場所: ノワール公爵邸、食堂
- 登場: アリシア、父（公爵）、母、侍女
- 目的: 「もう頑張らない」の具体的宣言。家族の困惑をコメディで描く
- 感情の弧: 開き直り → 少しだけ清々しい → 家族の反応に「あれ？」
- 描写ポイント:
  - 豪華すぎる朝食（7回目でも量は変わらない）
  - パンにバターを雑に塗るアリシア（令嬢失格の動作）
  - 「今日から礼儀作法の授業はお休みしますわ」→ 父の困惑顔
  - 「ドレスは楽なものだけで結構ですわ」→ 侍女の動揺
  - 母の「あらあら、いいんじゃないかしら」（意外と寛容）
  - 父の反応がこれまでの周回と違う（怒らない。心配する）→ アリシア「（え、怒らないの？　1周目は怒鳴られたんだけど……）」
- 伏線: 父の反応の違いが「7周目のズレ」の最初の兆候

### シーン4: 引き（アリシアの部屋）

- 場所: 部屋に戻ったアリシア
- 登場: アリシア、（声のみ）カイル
- 目的: 第1話の締め。カイルの登場予告で感情に刺す
- 感情の弧: 脱力 → カイルの声に胸が痛む → 「泣かない。7周目は泣かない」
- 描写ポイント:
  - ベッドにダイブするアリシア。「（7回目……はぁ）」
  - 天井を見上げていると、窓の外から声
  - 「おーい、アリシア！ 遊ぼうぜ！」
  - アリシアの独白:「——カイル。どの周回でも、最初に会いに来るのはこいつだ」
  - 6回分の記憶がフラッシュバック。「この男だけは、一度も私を裏切らなかった」
  - 「（……泣かない。7周目は、泣かないって決めた）」
  - 窓を開けるか迷うアリシアの手で終わり

## フック設計

### 冒頭フック
「——7回目の朝日は、やっぱり眩しかった。」
→ 「7回目」のインパクトで読者の「？」を引き出す。すぐにループ設定が明かされる

### 中盤転換（シーン3）
父の反応がこれまでと違う。読者はまだ気づかないが、アリシアが「あれ？」と引っかかる。
→ 「7周目のズレ」の最初の種

### 末尾の引き
カイルの声。6回分の記憶。泣かないと決めた涙。
→ コメディで始まった第1話が、最後の5行で感情に刺す。次話への期待を「笑い」から「感情」にシフト

## 読者体験の目標

- **笑い**: 6回のループ失敗ダイジェストで「この主人公面白い！」と思わせる
- **共感**: 「もう頑張りたくない」という気持ちに共感させる（現代の読者に刺さるテーマ）
- **驚き**: ループもの×悪役令嬢の組み合わせ。「頑張らない」という予想外の選択
- **引き**: カイルの存在で「この人はどんな人？　次が読みたい」と思わせる
