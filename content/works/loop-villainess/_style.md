# 文体プロファイル：7回目の人生で、悪役令嬢はもう頑張らないと決めた

## パラメータ

```yaml
pov: first_person              # 一人称（アリシア視点）
tempo: fast                     # 基本テンポ: 速め（コミカル、テンポ重視）
sentence_length_avg: 18         # 短文主体。切れ味重視
dialogue_ratio: 0.35            # 会話3.5割
inner_monologue_ratio: 0.35     # 内面独白3.5割（ツッコミ多め）
description_density: low        # 描写は最低限。テンポ優先
humor_level: very_high          # コミカル度最高。ギャグとシリアスの落差で笑わせる
tension_curve: wave             # コメディとシリアスの波
vocabulary_level: casual        # カジュアル寄り
line_break_frequency: frequent  # 改行多め（スマホ読み意識）
```

## 文体の特徴

この作品の文体は「疲れ切った主人公のツッコミ」が最大の武器。6回のループで蓄積した疲労と達観が、異世界のお約束をメタ的に笑い飛ばす。

- 地の文: アリシアの一人称。疲労感とツッコミが混在
- 独白: （）で囲む。投げやりだが的確なツッコミ
- 会話: 周囲のキャラはテンプレ的に振る舞う→アリシアの反応がズレる→ギャップで笑い
- シリアスとの落差: コメディの中にふと差し込まれる本音が泣ける

## 話ごとのテンポ目安

- 第1話: fast（7回目覚醒。「もう頑張らない」宣言。テンポよく）
- 第2-4話: fast（自由行動開始。周囲の困惑。コメディ全開）
- 第5-7話: fast→medium（周囲が変わり始める。アリシアの戸惑い）
- 第8-12話: medium（関係性の深化。シリアスが混ざり始める）
- 第13-18話: medium→fast（ループの真相に迫る）
- 第19-25話: fast→climax（最終決戦と大団円）
