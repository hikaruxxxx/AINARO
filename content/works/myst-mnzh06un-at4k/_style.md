# 文体プロファイル：逃亡刑事・槙島隼人

## パラメータ

```yaml
pov: third_person
tempo: fast
sentence_length_avg: 18
dialogue_ratio: 0.15
inner_monologue_ratio: 0.15
description_density: medium
humor_level: low
tension_curve: wave
vocabulary_level: standard
line_break_frequency: frequent
```

## 文体の特徴

短文を畳みかける硬質な文体。五感描写（雨の冷たさ、鉄の匂い、泥水の味）が身体感覚に直結する。刑事の思考プロセスを秒単位で描写し、過去回想と現在進行形を往復。体言止めと一行改行で緊張を刻む。内面独白と情景描写の比重が高く、冷徹な観察眼が全篇を貫く。
