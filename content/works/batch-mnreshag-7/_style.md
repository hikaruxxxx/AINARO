# 文体プロファイル：飾り婚約者と無口な騎士

```yaml
pov: first_person
tempo: slow

description_density: high
humor_level: low

line_break_frequency: frequent
```

## 文体の特徴

一人称視点で内面独白と情景描写を精密に織り交ぜる文体。五感（香水、蝋燭、石壁の匂い、硝子の冷たさ）を多用し、感情を直接語らず所作や呼吸で示す抑制的な筆致。短い体言止めと「——」を挟んだ心情の断ち切りが印象的で、会話より地の文の比重が高い。比喩は古典的で優雅。
