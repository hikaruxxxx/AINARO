# 文体プロファイル：凍れる手のひら

```yaml
pov: third_person
tempo: slow
sentence_length_avg: 28
dialogue_ratio: 0.15
inner_monologue_ratio: 0.15
description_density: high
humor_level: low
tension_curve: wave
vocabulary_level: standard
line_break_frequency: frequent
```

## 文体の特徴

短文の積み重ねによる抑制的な文体。感情を直接語らず、動作・音・匂いなど五感の細部で心理を描写する。「じゅう、と油が弾ける」「とん、とん、とん」など音の描写が効果的。内省は長くなりすぎず、行動描写に戻る呼吸がある。ダッシュによる挿入で時間軸を自在に往復する。
