# 文体プロファイル：星喰いの匙と禁じられた出汁

## パラメータ

```yaml
pov: third_person
tempo: slow
sentence_length_avg: 28
dialogue_ratio: 0.12
inner_monologue_ratio: 0.15
description_density: high
humor_level: low
tension_curve: wave
vocabulary_level: standard
line_break_frequency: frequent
```

## 文体の特徴

五感描写、特に味覚・嗅覚・触覚の描写が極めて緻密。比喩表現が多層的で詩的（「祈りのように」「氷の下で魚が動く音」等）。短文と長文を交互に配し、緊張と余韻を操作する。内面独白が厚く、感覚の微細な変化を逐一言語化する内省型の文体。
