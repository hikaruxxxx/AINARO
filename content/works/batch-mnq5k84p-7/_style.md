# 文体プロファイル：信長の膳番──戦国料理道

## パラメータ

```yaml
pov: third_person
tempo: slow
sentence_length_avg: 28
dialogue_ratio: 0.10
inner_monologue_ratio: 0.15
description_density: high
humor_level: low
tension_curve: wave
vocabulary_level: standard
line_break_frequency: frequent
```

## 文体の特徴

五感描写が極めて濃密。匂い・触感・温度を執拗に書き込み、読者の体感に訴える文体。短文と長文を交互に配置するリズム感があり、内面独白は断片的・詩的に挿入される。比喩表現が具体的で料理人の視点に根差しており、感傷に流れすぎず抑制が効いている。
