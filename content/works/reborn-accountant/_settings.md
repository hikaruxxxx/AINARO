# 帳簿使いは異世界を立て直す

## 基本情報

- ジャンル: 異世界転生ファンタジー（経済無双）
- 想定話数: 30話
- 構造: 連続型
- ターゲット: 20-30代男女
- ログライン: 過労死した経理部長が異世界の破産寸前の辺境領に転生。チート能力は「複式簿記」——剣も魔法も使えないが、領地の帳簿を立て直し、経済戦争で大国に挑む

## 差別化要素

- チート能力が「複式簿記」。戦闘ではなく経済で無双する
- 敵との対決も剣ではなく帳簿と交渉
- 主人公が「数字にしか興味がない」冷徹な人間から変わっていく成長譚

## 世界観

### 舞台

- 世界名: アルドラ大陸
- 時代感: 中世ヨーロッパ後期〜近世初期（商業革命期に相当）
- 通貨: クラウン金貨・銀貨・銅貨（1金貨 = 100銀貨 = 10,000銅貨）

### クランツ辺境領

- 大陸北東端の辺境領。鉱山資源（鉄・銀）が主産業
- 人口約8,000人。領都フロストヘルムに3,000人
- 財政状況: 破産寸前。年間歳入2,000金貨に対し歳出3,200金貨。累積債務4,500金貨
- 原因: 鉱山の産出量低下、不利な交易条件、杜撰な帳簿管理、中間搾取

### 王国と経済体制

- ヴァイゼン王国に属する辺境領のひとつ
- 帳簿は単式簿記（入金・出金の記録のみ）が一般的
- 大商会が流通を独占し、辺境領は不利な価格で取引させられている
- 「数字を扱う者」は軽んじられる文化。剣と魔法が尊ばれる

### 魔法体系

- 魔法は存在するが希少。魔法使いは人口の2-3%
- 主人公は魔力ゼロ。一切の魔法を使えない
- 魔法は戦闘・治療・農業補助が主用途。経済分野への応用はほぼない

## メインキャラクター（概要）

### トール・クランツ（主人公）

- 前世: 黒崎透（42歳）。東証一部上場企業の経理部長。過労死
- 転生先: クランツ辺境伯家の三男。18歳
- 外見: 黒髪、細身、眼鏡（この世界では珍しい矯正具）。目つきが鋭い
- 性格: 冷静沈着、数字への執着が異常。人間関係に不器用
- ギャップ: 外面は穏やかで頼りない三男坊 / 内面は冷徹な数字の鬼
- 口調: 丁寧語ベース。数字の話になると早口で専門用語が混じる
- 癖: 無意識に指で空中に数字を書く。考え事のとき眼鏡を押し上げる
- チート: 「複式簿記」の概念を持っている。帳簿を見れば不正・無駄が即座にわかる
- 変化の弧: 数字だけの人間 → 数字の向こうに人の暮らしを見る人間へ

### エルザ・フェルトハイム（ヒロイン）

- 役割: 徴税官の娘。実質的な領地の帳簿管理者
- 外見: 亜麻色の髪を一つ結び。インク染みのついた指。そばかす
- 性格: 真面目で責任感が強い。領地への愛着が深いが、現状に諦めかけている
- 口調: きっぱりとした丁寧語。数字の話では容赦なく厳しい
- 癖: 計算するとき小声でつぶやく。羽根ペンを耳に挟む
- 変化の弧: 諦観 → トールの手法に触れて希望を取り戻す → 自ら経済改革の担い手になる

### ヴィクトル・クランツ（父）

- 役割: クランツ辺境伯。トールの父
- 外見: 大柄、赤毛に白髪交じり。古傷が多い歴戦の武人
- 性格: 豪快で情に厚い。武人としては一流だが経済は苦手
- 口調: 豪快な男言葉。「〜だろう」「〜ではないか」
- 変化の弧: 「帳簿など武人の仕事ではない」 → 息子の才能を認め、任せる決断をする

### マルクス・ゴルトシュミット（敵役）

- 役割: 王都の大商会「黄金天秤商会」の会頭
- 外見: 恰幅のいい中年。金の指輪を複数。常に微笑んでいる
- 性格: 狡猾で計算高い。利益のためなら手段を選ばない
- 口調: 慇懃無礼。「これはこれは」「ご冗談を」と丁寧だが見下している
- 変化の弧: 辺境を舐めている → トールの手腕に焦り始める → 本気で潰しにかかる
