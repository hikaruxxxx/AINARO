# 文体プロファイル：万象崩壊の社畜勇者

## パラメータ

```yaml
pov: first_person
tempo: fast
sentence_length_avg: 18
dialogue_ratio: 0.25
inner_monologue_ratio: 0.35
description_density: medium
humor_level: medium
tension_curve: ascending
vocabulary_level: casual
line_break_frequency: frequent
```

## 文体の特徴

- 一人称視点で社畜経験を異世界に重ねる独白が多く、「就職面接と何が違う」「内定自慢」など現代社会への皮肉がブラックユーモアとして機能している。
- 体言止め・短文の連打でテンポを作り、行動と内省を交互に挟むリズム。「了解」「振り返らない」など一語文で感情を切る技法が特徴的。
- 五感描写を要所に挿入し（蛍光灯の白い光、血の鉄錆の味、石畳の冷たさ）、没入感を高めつつも冗長にならない密度感。
- ダッシュ（——）を多用して内面の断絶・転換を表現。思考の飛躍と現実認識のギャップを演出する。
- 社畜メタファーが世界観説明を兼ねており、「ブラック企業と王国の違いは玉座かオフィスチェアか」のように設定開示を主人公の性格描写と一体化させている。
