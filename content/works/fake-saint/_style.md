# 文体プロファイル：偽聖女は今日も神託を捏造する

## パラメータ

```yaml
pov: first_person              # 一人称（マーガレット視点）
tempo: fast                     # 基本テンポ: 速め（コメディ全振り）
sentence_length_avg: 18         # 短文主体。テンポとキレ重視
dialogue_ratio: 0.40            # 会話4割（ハッタリの会話劇が多い）
inner_monologue_ratio: 0.30     # 内面独白3割（ツッコミとハッタリの心理戦）
description_density: low        # 描写は最低限。勢いで読ませる
humor_level: very_high          # コメディ全振り。シリアスは後半にじわじわ
tension_curve: escalation       # ハッタリがどんどんエスカレートする構造
vocabulary_level: casual        # カジュアル寄り。聖女モード時のみ格式高い語彙
line_break_frequency: frequent  # 改行多め（スマホ読み意識）
```

## 文体の特徴

この作品の文体は「聖女の仮面」と「庶民の本音」の二面性で笑わせる。test-villainessの「令嬢と元OL」の構造に近いが、こちらは「嘘が上手い」ことが笑いの核。

- 地の文: マーガレットの一人称。焦りと計算が混在する
- 独白: （）で囲む。心臓バクバクの本音。嘘のアドリブ過程が丸見え
- 会話: 聖女モードの荘厳な台詞と、心の中の悲鳴のギャップ
- ハッタリ描写: マーガレットが即興で神託を捏造する過程を実況中継的に描く

## 話ごとのテンポ目安

- 第1話: fast（聖女認定の事故。テンポよくコメディ開幕）
- 第2-4話: fast（初めての神託。ハッタリの連続。コメディ全開）
- 第5-7話: fast（ハッタリが結果的に成功し、期待値が上がる地獄）
- 第8-12話: fast→medium（セドリックの真意。マーガレットの葛藤が混ざる）
- 第13-18話: medium（偽聖女の噂。追い詰められていく）
- 第19-25話: medium→climax（真実の公開。偽物の聖女が起こす本物の奇跡）
