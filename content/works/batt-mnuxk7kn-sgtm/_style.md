# 文体プロファイル：夜衛の穴、十五の拳

```yaml
pov: first_person

```

## 文体の特徴

一人称の独白が中心で、身体感覚（血の味、肋骨の痛み、腐臭、冷気）を克明に積み上げる描写主体。短文を重ねて雨音や靴音のリズムで場面を刻む。歌舞伎町の固有名詞とネオン・蛍光灯の光の対比、体温と路地の温度差で孤独を浮かび上がらせる。比喩は控えめで、感覚の即物的な記述から少年の諦観を滲ませる文体。
