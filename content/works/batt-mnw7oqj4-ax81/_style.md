# 文体プロファイル：神経叢射のプレクサス

## パラメータ

```yaml
pov: first_person
tempo: slow
sentence_length_avg: 28
dialogue_ratio: 0.10
inner_monologue_ratio: 0.30
description_density: high
humor_level: low
tension_curve: wave
vocabulary_level: standard

```

## 文体の特徴

一人称で主人公の内面を精密に描写する硬質な文体。感情を持たない少年の視点から、五感の情報を淡々と列挙することで逆説的に読者の感情を揺さぶる技法。短文の積み重ねによるリズム感と、句読点の間合いで緊張感を制御している。
