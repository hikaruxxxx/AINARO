# 文体プロファイル：壁のシミは物語を書く

## パラメータ

```yaml
pov: first_person
tempo: slow
sentence_length_avg: 28
dialogue_ratio: 0.05
inner_monologue_ratio: 0.30
description_density: high
humor_level: low
tension_curve: wave
vocabulary_level: standard
line_break_frequency: frequent
```

## 文体の特徴

一人称の内省的な語りで、観察と思考が交互に連なる。比喩が繊細で具体的（「空気」ではなく「壁のシミ」）。推敲の過程を地の文に織り込む独特のメタ構造。短文と長文のリズム変化で静けさを演出。五感描写が豊かで、特に音と匂いへの感度が高い。
